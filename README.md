# CounterStrike2D #

## Instalar CMake ##
- sudo apt-get install cmake

## Instalar SDL2 ##
- sudo apt-get install libsdl2-dev
- sudo apt-get install libsdl2-image-dev
- sudo apt-get install libsdl2-mixer-dev

## Instalar Box2D ##
- sudo apt-get install libbox2d-dev

## Build ##
- mkdir build
- cd build
- cmake ..
- cmake --build .
